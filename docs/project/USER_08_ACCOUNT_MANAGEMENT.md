## Profile Management

### View Profile

**Information Displayed:**
- Name
- Phone Number (login ID)
- Email (optional)
- Account Created Date

### Edit Profile

**Editable Fields:**
- Name (letters and spaces only)
- Email address (optional)

**Steps:**
1. Click "Edit Profile"
2. Modify fields
3. Click "Save Changes"

**Validation:**
- Name: 2-50 characters, letters only
- Email: Valid format (if provided)

### Add New Address

**Required Fields:**
- Full Name *
- Phone Number * (10 digits)
- House/Flat/Building *
- Area/Locality *
- City *
- Pincode * (6 digits)
- Type: Home / Office / Other

**Optional:** Landmark, delivery instructions

### Edit Address

1. Click "Edit" on address card
2. Modify fields
3. Save changes

### Set Default Address

- Click "Set as Default"
- Used automatically at checkout

### Delete Address

- Click "Delete"
- Confirm deletion
- Cannot be undone

## Security & Privacy

### Change Phone Number

**Process:**
1. Contact customer support
2. Verify identity
3. Update phone number
4. All data migrated to new number

**Note:** Phone number is your login ID.

### Account Deletion

**Request account deletion:**
1. Contact support
2. Verify identity
3. Account deleted within 30 days
4. All data permanently removed

**Warning:** Cannot be undone.

### Privacy Settings

**Data Sharing:**
- Marketing emails: Opt-in/out
- SMS notifications: Opt-in/out
- Data used per [Privacy Policy](../PrivacyPolicy.md)

## Troubleshooting

**Cannot update profile:**
- Check field validation
- Ensure proper format
- Contact support if error persists

**Address not saving:**
- Fill all required fields (*)
- Verify pincode format (6 digits)
- Check phone number (10 digits)

**Cannot login:**
- Verify phone number
- Check OTP expiry (5 minutes)
- Request new OTP
- Contact support if blocked
