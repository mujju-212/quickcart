## 📱 Creating Your Account

### Step 1: Access Login

1. Click **"Login"** button in the top navigation bar
2. Or click the user icon in the header

### Step 2: Choose Login Method

You have two options:

#### Option A: Phone Number Login (Recommended)

1. Select **"Phone Login"** tab
2. Enter your 10-digit mobile number
3. Click **"Send OTP"** button

**Phone Number Format:**
```
Correct: 9876543210
Correct: +919876543210
Incorrect: 987-654-3210
```

4. Check your phone for the OTP (One-Time Password)
5. Enter the 6-digit OTP in the verification modal
6. Click **"Verify OTP"**

**OTP Details:**
- Valid for 10 minutes
- Can resend after 60 seconds
- Maximum 20 OTP requests per day per number

#### Option B: Email Login

1. Select **"Email Login"** tab
2. Enter your email address
3. Click **"Send OTP"**
4. Check your email inbox for OTP
5. Enter OTP and verify

### Step 3: Complete Your Profile

After first-time OTP verification:

1. **Profile Completion Modal** will appear
2. Fill in required information:
   - **Full Name** (required)
   - **Email** (optional but recommended)
   - **Phone** (pre-filled)

3. Click **"Complete Profile"**

**Profile Tips:**
- Use your real name for delivery purposes
- Add email for order confirmations
- Keep phone number accurate for delivery updates

## ⚡ Quick Start Guide

### Your First Order in 5 Steps

#### Step 1: Browse Products
1. Scroll through homepage categories
2. OR click a category to explore
3. OR use search bar to find specific items

**Example Search:**
- Search: "tomato" → Shows all tomato products
- Search: "dairy" → Shows milk, cheese, yogurt, etc.

#### Step 2: Add to Cart
1. Click on a product card
2. View product details
3. Click **"Add to Cart"** button
4. Choose quantity using +/- buttons
5. Continue shopping or proceed to checkout

**Product Card Information:**
- Product image
- Product name
- Size/weight
- Current price (discounted price if on sale)
- Original price (strikethrough if discounted)
- Stock status
- Add to Cart button

#### Step 3: Review Your Cart
1. Click cart icon in header
2. Review selected items
3. Adjust quantities (+/-)
4. Remove unwanted items (🗑️ icon)
5. View cart summary

**Cart Summary Shows:**
- Subtotal (sum of all items)
- Delivery fee (₹29, free on orders ≥ ₹99)
- Discount (if coupon applied)
- **Total Amount**

#### Step 4: Apply Coupon (Optional)
1. In cart page, find "Apply Coupon" section
2. Enter coupon code (e.g., FIRST20)
3. Click **"Apply"**
4. Discount will be reflected in total

**Coupon Rules:**
- Only one coupon per order
- Check minimum order value
- Valid coupons shown on homepage

#### Step 5: Checkout & Place Order
1. Click **"Proceed to Checkout"**
2. Verify delivery address (or add new)
3. Select payment method
4. Review order summary
5. Click **"Place Order"**

**Checkout Information Required:**
- Delivery address
- Contact phone number
- Payment method selection

## 🎯 Key Features Overview

### 1. Product Search

**How to Search:**
1. Click search bar in header
2. Type product name or category
3. Press Enter or click search icon
4. View search results

**Search Tips:**
- Be specific: "red apple" vs "apple"
- Use category names: "beverages"
- Try partial names: "tom" finds "tomatoes"

### 2. Wishlist

Save products for later:

1. Click ❤️ icon on product card
2. Access wishlist from account menu
3. View all saved items
4. Quick add to cart from wishlist

**Wishlist Benefits:**
- Save favorite products
- Quick reordering
- Track price changes
- Share with family

### 3. Multiple Addresses

Manage delivery locations:

1. Go to **Account** → **Addresses**
2. Click **"Add New Address"**
3. Fill in address details
4. Set as default (optional)
5. Save address

**Address Types:**

**Tracking Steps:**
1. Go to **Account** → **My Orders**

4. See current status

## 🔧 Troubleshooting

### Common Issues & Solutions

#### Issue 1: OTP Not Received

**Solutions:**
1. Check phone number is correct
2. Wait 2-3 minutes
3. Check SMS inbox and spam
4. Click "Resend OTP" after 60 seconds
5. Ensure good network connection

**Still not working?**
- Try email OTP instead
- Contact support

#### Issue 2: Cannot Add to Cart

**Possible Reasons:**
1. Not logged in → Login first
2. Product out of stock → Check stock status
3. Quantity exceeds stock → Reduce quantity

#### Issue 3: Coupon Not Working

**Check these:**
- Minimum order value met?
- Coupon still valid (check dates)?
- Only one coupon can be used
- Spelling is correct
- Coupon applicable to your items

#### Issue 4: Address Not Saving

**Solutions:**
1. Fill all required fields
2. Check internet connection
3. Try again after refresh
4. Clear browser cache

#### Issue 5: Payment Failed

**Steps:**
1. Check payment method details
2. Ensure sufficient balance/credit
3. Try alternative payment method
4. Contact bank if issue persists
