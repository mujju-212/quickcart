\section*{3.5 Technical Requirements}

\textbf{Architecture:} Three-tier architecture (presentation, business logic, data layers), RESTful API design, stateless backend with JWT session management, component-based React frontend.

\textbf{Database:} PostgreSQL with normalized schema (3NF), indexed columns for query optimization, foreign key constraints, trigger-based timestamp updates.

\textbf{Security:} HTTPS protocol, JWT with HS256 algorithm, CSRF token protection, input validation (Bleach library), parameterized queries for SQL injection prevention, rate limiting, security headers (X-Frame-Options, X-Content-Type-Options).

\textbf{API \& Integration:} RESTful endpoints with JSON payloads, Bearer token authentication, CORS configuration, SMS gateway integration (Twilio/Fast2SMS), PDF generation (jsPDF).
